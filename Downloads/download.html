Marvel’s Spider-Man Update Version 1.07 Patch Notes, Adds New Game Plus And More

October 16, 2018
Add comment
Khurram Imtiaz
Facebook Twitter Reddit Google+

Marvel’s Spider-Man will get a new update that will patch the game to version 1.07. Insomniac has confirmed some of the new additions that will be a part of this new update.

Recently, the game director, Bryan Intihar confirmed that Marvel’s Spider-Man will get a new game plus mode ‘soon’ and now it appears that they are finally adding it as part of a new update on October 17th.

Marvel’s Spider-Man Update Version 1.07 Patch Notes For PS4

Insomniac Games has confirmed some of the changes that will be a part of this new update on PS4.

  • Adds New Game+ (NG+) mode.
  • Adds Ultimate Difficulty mode.
  • Add two new trophies to the game.
  • Adds the ability to rotate Stickers in Photo Mode.
  • Tweaks to the Friendly Difficulty to make it more friendly.
  • Additional bug fixes.

That’s it for now. There will be a full set of patch notes available once the update goes live. In the meanwhile, you can find out the changes from the patch notes listed above. Don’t expect them to be the only ones, as Insomniac Games has usually shared a complete set of bug fixes after every update for Marvel’s Spider-Man.

Marvel’s Spider-Man is available now exclusively for the PS4. It was developed by Insomniac Games and published by Sony Interactive Entertainment.
